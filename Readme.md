## Anot 框架
> `Anot` 是`Anot not only templateEngine`的缩写。
> 它是一款迷你,易用、高性能的前端MVVM框架, fork于avalon。进行了大量的重构,精简部分冗余的API, 同时针对组件拓展进行了优化。


## doUI 组件库
> doUI组件库是基于`Anot`框架开发的一套高效,轻量,可定制的现代化组件库。


## 开发环境及生产环境
```bash
# 开发环境
npm start

# 生产环境
npm run build
```
